Search any Tailwind CSS class to see its CSS output. Inspect elements to discover Tailwind classes on any page.

## Features

- Inspect any element to see its Tailwind classes
- Search any Tailwind class to see the generated CSS
- Reverse lookup: paste CSS to find the matching Tailwind utility
- Works with Tailwind v3 and v4 classes
- Light and dark themes
