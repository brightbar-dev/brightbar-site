Auto-detects and formats JSON responses with an interactive tree view, search, and syntax highlighting.

## Features

- Auto-detects JSON responses and formats them in-page
- Collapsible tree view for navigating nested structures
- Full-text search across keys and values
- Syntax highlighting with light and dark themes
- Copy paths, values, or formatted JSON
