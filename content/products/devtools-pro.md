12 essential web development tools in one extension. No juggling multiple tools, no switching contexts.

## Free Tools

Use without limits, forever.

- **CSS Inspector** — View computed styles for any element, organized by category
- **Color Picker** — Pick any color on the page, copy as hex/RGB/HSL, see WCAG contrast ratios
- **Font Detector** — Identify fonts, weights, sizes, and line heights on any element
- **Spacing Visualizer** — See margins, padding, and borders with a visual box model diagram
- **Element Info** — Tag name, classes, dimensions, position, and a unique CSS selector
- **Page Meta** — All meta tags at a glance: Open Graph, Twitter cards, viewport, SEO tags

## Pro Tools

$60 one-time purchase. 7-day free trial.

- **Screenshot Capture** — Capture the visible viewport as PNG
- **Accessibility Checker** — Audit headings, landmarks, ARIA roles, alt text, form labels
- **CSS Variables** — Extract every CSS custom property, grouped by scope
- **Rulers & Measurement** — Hover to see element dimensions and distances
- **Grid & Flexbox Overlay** — Visualize grid and flexbox layouts with container details
- **Page Assets** — List all images, scripts, stylesheets, and fonts on the page
