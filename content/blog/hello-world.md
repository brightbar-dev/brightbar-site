We're Bright Bar, and we build developer tools that respect your privacy.

Every extension we ship is open source, collects zero user data, and does exactly what it says on the tin — nothing more.

## What we've built so far

**[JSON Viewer Pro](/products/)** — Auto-detects and formats JSON responses with an interactive tree view, search, and themes. Available now on the Chrome Web Store.

**[Tailwind CSS Lookup](/products/)** — Inspect Tailwind classes on any element. Search any class to see the CSS it generates.

**[Cookie DevTools](/products/)** — A developer-focused cookie manager with real-time monitoring, profiles, and export to curl/wget.

## What to expect here

This blog will cover what we're building, what we've learned along the way, and occasional deep dives into browser extension development. No fluff, no growth hacking, no engagement bait.
